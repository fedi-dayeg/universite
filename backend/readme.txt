Before installing, download and install 
	1- Node.js LTS version https://nodejs.org/en/download
	2- Mongodb Version 4 or higher  https://www.mongodb.com/download-center/community

Then Installation
 I- back-end server
#technology used : using nodejs
install all dependencies by ==> back-end# npm install
run the server by ==> back-end# npm start
you should see server running...  at http://localhost:4000  or http://IP:4000

 II- admin-lte-dashboard
#technology used : using react js framework
install all dependencies by ==> admin-lte-dashboard# npm install
run the Application by ==> admin-lte-dashboard# npm start
you should see server running...  at http://localhost:3000  or http://IP:3000
